SpringCloud GatewayдҪңдёәSpring Cloud з”ҹжҖҒзі»з»ҹдёӯзҡ„зҪ‘е…іпјҢзӣ®ж ҮжҳҜжӣҝд»ЈZuulпјҢеңЁSpring Cloud 2.0д»ҘдёҠзүҲжң¬дёӯпјҢжІЎжңүеҜ№ж–°зүҲжң¬зҡ„Zul 2.0д»ҘдёҠжңҖж–°й«ҳжҖ§иғҪзүҲжң¬иҝӣиЎҢйӣҶжҲҗпјҢд»Қз„¶иҝҳжҳҜдҪҝз”Ёзҡ„Zuul 1.xйқһReactorжЁЎејҸзҡ„иҖҒзүҲжң¬гҖӮиҖҢдёәдәҶжҸҗеҚҮзҪ‘е…ізҡ„жҖ§иғҪпјҢSpringCloud GatewayжҳҜеҹәдәҺWebFluxжЎҶжһ¶е®һзҺ°зҡ„пјҢиҖҢWebFluxжЎҶжһ¶еә•еұӮеҲҷдҪҝз”ЁдәҶй«ҳжҖ§иғҪзҡ„ReactorжЁЎејҸйҖҡдҝЎжЎҶжһ¶NettyгҖӮ

# GateWayйқһйҳ»еЎһејӮжӯҘжЁЎеһӢ

жңүZuullдәҶжҖҺд№ҲеҸҲеҮәжқҘGatewayпјҹжҲ‘д»¬дёәд»Җд№ҲйҖүжӢ©Gateway?

  1. netflixдёҚеӨӘйқ и°ұпјҢzuul2.0дёҖзӣҙи·ізҘЁпјҢиҝҹиҝҹдёҚеҸ‘еёғгҖӮ

    1. дёҖж–№йқўеӣ дёәZuul1.0е·Із»Ҹиҝӣе…ҘдәҶз»ҙжҠӨйҳ¶ж®өпјҢиҖҢдё”GatewayжҳҜSpringCloudеӣўйҳҹз ”еҸ‘зҡ„пјҢжҳҜдәІе„ҝеӯҗдә§е“ҒпјҢеҖјеҫ—дҝЎиө–гҖӮиҖҢдё”еҫҲеӨҡеҠҹиғҪZuulйғҪжІЎжңүз”Ёиө·жқҘд№ҹйқһеёёзҡ„з®ҖеҚ•дҫҝжҚ·гҖӮ
    2. GatewayжҳҜеҹәдәҺејӮжӯҘйқһйҳ»еЎһжЁЎеһӢдёҠиҝӣиЎҢејҖеҸ‘зҡ„пјҢжҖ§иғҪж–№йқўдёҚйңҖиҰҒжӢ…еҝғгҖӮиҷҪз„¶Netflixж—©е°ұеҸ‘еёғдәҶжңҖж–°зҡ„Zuul 2.xпјҢдҪҶSpring CloudиІҢдјјжІЎжңүж•ҙеҗҲи®ЎеҲ’гҖӮиҖҢдё”Netflixзӣёе…із»„д»¶йғҪе®Јеёғиҝӣе…Ҙз»ҙжҠӨжңҹпјӣдёҚзҹҘеүҚжҷҜеҰӮдҪ•?
    3. еӨҡж–№йқўз»јеҗҲиҖғиҷ‘GatewayжҳҜеҫҲзҗҶжғізҡ„зҪ‘е…ійҖүжӢ©гҖӮ

  2. SpringCloud Gatewayе…·жңүеҰӮдёӢзү№жҖ§

    1. еҹәдәҺSpring Framework 5пјҢProject Reactorе’ҢSpring Boot 2.0иҝӣиЎҢжһ„е»әпјӣ
    2. еҠЁжҖҒи·Ҝз”ұпјҡиғҪеӨҹеҢ№й…Қд»»дҪ•иҜ·жұӮеұһжҖ§пјӣ
    3. еҸҜд»ҘеҜ№и·Ҝз”ұжҢҮе®ҡPredicate (ж–ӯиЁҖ)е’ҢFilter(иҝҮж»ӨеҷЁ)пјӣ
    4. йӣҶжҲҗHystrixзҡ„ж–ӯи·ҜеҷЁеҠҹиғҪпјӣ
    5. йӣҶжҲҗSpring Cloud жңҚеҠЎеҸ‘зҺ°еҠҹиғҪпјӣ

    1. еңЁSpringCloud FinchleyжӯЈејҸзүҲд№ӢеүҚпјҢSpring CloudжҺЁиҚҗзҡ„зҪ‘е…іжҳҜNetflixжҸҗдҫӣзҡ„ZuulгҖӮ
    2. Zuul 1.xпјҢжҳҜдёҖдёӘеҹәдәҺйҳ»еЎһI/Oзҡ„API GatewayгҖӮ
    3. Zuul 1.xеҹәдәҺServlet 2.5дҪҝз”Ёйҳ»еЎһжһ¶жһ„е®ғдёҚж”ҜжҢҒд»»дҪ•й•ҝиҝһжҺҘ(еҰӮWebSocket)Zuulзҡ„и®ҫи®ЎжЁЎејҸе’ҢNginxиҫғеғҸпјҢжҜҸж¬ЎI/Рһж“ҚдҪңйғҪжҳҜд»Һе·ҘдҪңзәҝзЁӢдёӯйҖүжӢ©дёҖдёӘжү§иЎҢпјҢиҜ·жұӮзәҝзЁӢиў«йҳ»еЎһеҲ°е·ҘдҪңзәҝзЁӢе®ҢжҲҗпјҢдҪҶжҳҜе·®еҲ«жҳҜNginxз”ЁC++е®һзҺ°пјҢZuulз”ЁJavaе®һзҺ°пјҢиҖҢJVMжң¬иә«дјҡжңүз¬¬-ж¬ЎеҠ иҪҪиҫғж…ўзҡ„жғ…еҶөпјҢдҪҝеҫ—Zuulзҡ„жҖ§иғҪзӣёеҜ№иҫғе·®гҖӮ
    4. Zuul 2.xзҗҶеҝөжӣҙе…ҲиҝӣпјҢжғіеҹәдәҺNettyйқһйҳ»еЎһе’Ңж”ҜжҢҒй•ҝиҝһжҺҘпјҢдҪҶSpringCloudзӣ®еүҚиҝҳжІЎжңүж•ҙеҗҲгҖӮZuul .xзҡ„жҖ§иғҪиҫғZuul 1.xжңүиҫғеӨ§жҸҗеҚҮгҖӮеңЁжҖ§иғҪж–№йқўпјҢж №жҚ®е®ҳж–№жҸҗдҫӣзҡ„еҹәеҮҶжөӢиҜ•,Spring Cloud Gatewayзҡ„RPS(жҜҸз§’иҜ·жұӮж•°)жҳҜZuulзҡ„1.6еҖҚгҖӮ
    5. Spring Cloud Gatewayе»әз«ӢеңЁSpring Framework 5гҖҒProject Reactorе’ҢSpring Boot2д№ӢдёҠпјҢдҪҝз”Ёйқһйҳ»еЎһAPIгҖӮ
