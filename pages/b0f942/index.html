# @RequestMappingжіЁи§Јзҡ„еҠҹиғҪ

д»ҺжіЁи§ЈеҗҚз§°дёҠжҲ‘д»¬еҸҜд»ҘзңӢеҲ°пјҢ@RequestMappingжіЁи§Јзҡ„дҪңз”Ёе°ұжҳҜе°ҶиҜ·жұӮе’ҢеӨ„зҗҶиҜ·жұӮзҡ„жҺ§еҲ¶еҷЁж–№жі•е…іиҒ”иө·жқҘпјҢе»әз«Ӣжҳ е°„е…ізі»гҖӮ

SpringMVC жҺҘж”¶еҲ°жҢҮе®ҡзҡ„иҜ·жұӮпјҢе°ұдјҡжқҘжүҫеҲ°еңЁжҳ е°„е…ізі»дёӯеҜ№еә”зҡ„жҺ§еҲ¶еҷЁж–№жі•жқҘеӨ„зҗҶиҝҷдёӘиҜ·жұӮгҖӮ

# @RequestMappingжіЁи§Јзҡ„дҪҚзҪ®

@RequestMappingж ҮиҜҶдёҖдёӘзұ»пјҡи®ҫзҪ®жҳ е°„иҜ·жұӮзҡ„иҜ·жұӮи·Ҝеҫ„зҡ„еҲқе§ӢдҝЎжҒҜ

@RequestMappingж ҮиҜҶдёҖдёӘж–№жі•пјҡи®ҫзҪ®жҳ е°„иҜ·жұӮиҜ·жұӮи·Ҝеҫ„зҡ„е…·дҪ“дҝЎжҒҜ

@Controller
@RequestMapping("/test")
public class RequestMappingController {
	//жӯӨж—¶иҜ·жұӮжҳ е°„жүҖжҳ е°„зҡ„иҜ·жұӮзҡ„иҜ·жұӮи·Ҝеҫ„дёәпјҡ/test/testRequestMapping
    @RequestMapping("/testRequestMapping")
    public String testRequestMapping(){
        return "success";
    }
}

# @RequestMappingжіЁи§Јзҡ„methodеұһжҖ§

@RequestMappingжіЁи§Јзҡ„methodеұһжҖ§йҖҡиҝҮиҜ·жұӮзҡ„иҜ·жұӮж–№ејҸпјҲgetжҲ–postпјүеҢ№й…ҚиҜ·жұӮжҳ е°„

@RequestMappingжіЁи§Јзҡ„methodеұһжҖ§жҳҜдёҖдёӘRequestMethodзұ»еһӢзҡ„ж•°з»„пјҢиЎЁзӨәиҜҘиҜ·жұӮжҳ е°„иғҪеӨҹеҢ№й…ҚеӨҡз§ҚиҜ·жұӮж–№ејҸзҡ„иҜ·жұӮ

  1. еҜ№дәҺеӨ„зҗҶжҢҮе®ҡиҜ·жұӮж–№ејҸзҡ„жҺ§еҲ¶еҷЁж–№жі•пјҢSpringMVCдёӯжҸҗдҫӣдәҶ@RequestMappingзҡ„жҙҫз”ҹжіЁи§Ј

    еӨ„зҗҶgetиҜ·жұӮзҡ„жҳ е°„-->@GetMapping

    еӨ„зҗҶpostиҜ·жұӮзҡ„жҳ е°„-->@PostMapping

    еӨ„зҗҶputиҜ·жұӮзҡ„жҳ е°„-->@PutMapping

    еӨ„зҗҶdeleteиҜ·жұӮзҡ„жҳ е°„-->@DeleteMapping

  2. еёёз”Ёзҡ„иҜ·жұӮж–№ејҸжңүgetпјҢpostпјҢputпјҢdelete

    дҪҶжҳҜзӣ®еүҚжөҸи§ҲеҷЁеҸӘж”ҜжҢҒgetе’ҢpostпјҢиӢҘеңЁformиЎЁеҚ•жҸҗдәӨж—¶пјҢдёәmethodи®ҫзҪ®дәҶе…¶д»–иҜ·жұӮж–№ејҸзҡ„еӯ—з¬ҰдёІпјҲputжҲ–deleteпјүпјҢеҲҷжҢүз…§й»ҳи®Өзҡ„иҜ·жұӮж–№ејҸgetеӨ„зҗҶ

    иӢҘиҰҒеҸ‘йҖҒputе’ҢdeleteиҜ·жұӮпјҢеҲҷйңҖиҰҒйҖҡиҝҮspringжҸҗдҫӣзҡ„иҝҮж»ӨеҷЁHiddenHttpMethodFilterпјҢеңЁRESTfulйғЁеҲҶдјҡи®ІеҲ°

# @RequestMappingжіЁи§Јзҡ„paramsеұһжҖ§пјҲдәҶи§Јпјү

@RequestMappingжіЁи§Јзҡ„paramsеұһжҖ§йҖҡиҝҮиҜ·жұӮзҡ„иҜ·жұӮеҸӮж•°еҢ№й…ҚиҜ·жұӮжҳ е°„

@RequestMappingжіЁи§Јзҡ„paramsеұһжҖ§жҳҜдёҖдёӘеӯ—з¬ҰдёІзұ»еһӢзҡ„ж•°з»„пјҢеҸҜд»ҘйҖҡиҝҮеӣӣз§ҚиЎЁиҫҫејҸи®ҫзҪ®иҜ·жұӮеҸӮж•°е’ҢиҜ·жұӮжҳ е°„зҡ„еҢ№й…Қе…ізі»

"param"пјҡиҰҒжұӮиҜ·жұӮжҳ е°„жүҖеҢ№й…Қзҡ„иҜ·жұӮеҝ…йЎ»жҗәеёҰparamиҜ·жұӮеҸӮж•°

"!param"пјҡиҰҒжұӮиҜ·жұӮжҳ е°„жүҖеҢ№й…Қзҡ„иҜ·жұӮеҝ…йЎ»дёҚиғҪжҗәеёҰparamиҜ·жұӮеҸӮж•°

"param=value"пјҡиҰҒжұӮиҜ·жұӮжҳ е°„жүҖеҢ№й…Қзҡ„иҜ·жұӮеҝ…йЎ»жҗәеёҰparamиҜ·жұӮеҸӮж•°дё”param=value

"param!=value"пјҡиҰҒжұӮиҜ·жұӮжҳ е°„жүҖеҢ№й…Қзҡ„иҜ·жұӮеҝ…йЎ»жҗәеёҰparamиҜ·жұӮеҸӮж•°дҪҶжҳҜparam!=value

# @RequestMappingжіЁи§Јзҡ„headersеұһжҖ§пјҲдәҶи§Јпјү

@RequestMappingжіЁи§Јзҡ„headersеұһжҖ§йҖҡиҝҮиҜ·жұӮзҡ„иҜ·жұӮеӨҙдҝЎжҒҜеҢ№й…ҚиҜ·жұӮжҳ е°„

@RequestMappingжіЁи§Јзҡ„headersеұһжҖ§жҳҜдёҖдёӘеӯ—з¬ҰдёІзұ»еһӢзҡ„ж•°з»„пјҢеҸҜд»ҘйҖҡиҝҮеӣӣз§ҚиЎЁиҫҫејҸи®ҫзҪ®иҜ·жұӮеӨҙдҝЎжҒҜе’ҢиҜ·жұӮжҳ е°„зҡ„еҢ№й…Қе…ізі»

"header"пјҡиҰҒжұӮиҜ·жұӮжҳ е°„жүҖеҢ№й…Қзҡ„иҜ·жұӮеҝ…йЎ»жҗәеёҰheaderиҜ·жұӮеӨҙдҝЎжҒҜ

"!header"пјҡиҰҒжұӮиҜ·жұӮжҳ е°„жүҖеҢ№й…Қзҡ„иҜ·жұӮеҝ…йЎ»дёҚиғҪжҗәеёҰheaderиҜ·жұӮеӨҙдҝЎжҒҜ

"header=value"пјҡиҰҒжұӮиҜ·жұӮжҳ е°„жүҖеҢ№й…Қзҡ„иҜ·жұӮеҝ…йЎ»жҗәеёҰheaderиҜ·жұӮеӨҙдҝЎжҒҜдё”header=value

"header!=value"пјҡиҰҒжұӮиҜ·жұӮжҳ е°„жүҖеҢ№й…Қзҡ„иҜ·жұӮеҝ…йЎ»жҗәеёҰheaderиҜ·жұӮеӨҙдҝЎжҒҜдё”header!=value

иӢҘеҪ“еүҚиҜ·жұӮж»Ўи¶і@RequestMappingжіЁи§Јзҡ„valueе’ҢmethodеұһжҖ§пјҢдҪҶжҳҜдёҚж»Ўи¶іheadersеұһжҖ§пјҢжӯӨж—¶йЎөйқўжҳҫзӨә404й”ҷиҜҜпјҢеҚіиө„жәҗжңӘжүҫеҲ°
